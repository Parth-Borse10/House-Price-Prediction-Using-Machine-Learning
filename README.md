# 🏠 House Price Prediction Using Machine Learning

## Overview

A full-stack Machine Learning web application that predicts house prices using property features and a Random Forest Regression model.

## Features

* House Price Prediction
* Random Forest Regression Model
* Flask REST API
* Interactive Frontend
* Model Performance Metrics
* Feature Importance Analysis

## Tech Stack

* Python
* Flask
* Scikit-Learn
* Pandas
* NumPy
* HTML
* CSS
* JavaScript

## Installation

```bash
pip install -r requirements.txt
python app.py
```
